# Word Finder

## Описание

Данный проект представляет собой многопоточное приложение для подсчета определенной буквы в тексте. Приложение выполняет следующие шаги:

1. Разбивает большой текстовый файл на блоки строк заданного размера.
2. Для каждого блока создается задача, которая выполняется в нескольких потоках.
3. Для синхронизации выполнения потоков используется барьер (CyclicBarrier).
4. Итоговый результат – общее количество заданной буквы в тексте – выводится после выполнения всех задач.

## Структура проекта

- **ComplexTask**: Класс для подсчета количества заданной буквы в строке.
- **TextFileSplitter**: Класс для разбиения текстового файла на блоки строк.
- **ComplexTaskExecutor**: Класс для выполнения задач в многопоточном режиме с использованием барьера для синхронизации.
- **TestComplexTaskExecutor**: Тестовый класс для запуска приложения и демонстрации его работы с использованием нескольких потоков.

## Запуск

1. Склонируйте проект на свою машину.
2. Убедитесь, что у вас установлен JDK 8+.
3. Поместите текстовый файл для анализа в папку `src/main/resources/text/`.
4. Запустите `TestComplexTaskExecutor` для выполнения программы.
5. Измените параметры программы (количество потоков, задач, строк на задачу) по необходимости.

## Пример использования

Пример запуска программы с файлом `pg2600.txt`, используя 5 потоков, где каждая задача анализирует 100 строк текста.
